# Информационная система учета организации питания обучающихся

Веб-версия для размещения на хостинге.

## Технологии

- HTML
- CSS
- JavaScript
- Python
- SQLite

## Локальный запуск

```bash
python server.py
```

После запуска открыть:

```text
http://localhost:3000
```

## Размещение на Render

1. Создать репозиторий на GitHub.
2. Загрузить в него все файлы из этого архива.
3. На Render выбрать New Web Service.
4. Подключить GitHub-репозиторий.
5. Указать:
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python server.py`
6. После деплоя приложение будет доступно по ссылке вида:
   `https://название-проекта.onrender.com`

## Важно

База SQLite создается автоматически при первом запуске.
Для учебного проекта этого достаточно. Для постоянной реальной эксплуатации лучше использовать серверную БД, например PostgreSQL.
